Overview of Game

Cops and Robbers is a very basic game based off of collision logic and movement logic. The idea of the game is for the user (the Robber) to avoid the Cop (the computer) for as long as possible. The Cop chases the Robber down for 10 seconds, and if the Robber can escape for 10 seconds they win. The Cop speeds up as time goes on, and the user can also set a speed of the Cop at the start of the game with the HTML text input. My inspiration for the game was the demo with the ellipse that follows the mouse around, as well as the game "Tag."

Documentation of Physical Controllers

There are many images and photos from the Makey Makey class group work.
